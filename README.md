This project explores Citibike usage patterns in New York City using data from March to May 2025. It analyzes ride activity by station, time of day, day of week, user type (member vs. casual), and bike type (classic vs. electric). Interactive visualizations highlight differences in behavior, station popularity, and opportunities for system optimization.

Tools used: Tableau, Excel

Data source: Citibike System Data
